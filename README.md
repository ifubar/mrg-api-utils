## Api utilities meetings Magora guidelines

__Installation:__

```shell
//composer.json
    "require": {
    	"ifubar/mrg-api-utils": "*"
    },
    "repositories": [
        { "type": "vcs", "url": "https://github.com/ifubar/mrg-api-utils" }
    ],
    "minimum-stability": "dev"
```

```shell
    composer install
```
